# Credits & Acknowledgements

Flux is made possible thanks to the following incredible open-source projects, APIs, and resources.

## Core Media Engine
- **[mpv](https://mpv.io/)**: The core media player engine (GPLv2+).
- **[FFmpeg](https://ffmpeg.org/)**: For decoding, encoding, transcoding, and filtering (LGPLv2.1+).
- **[libass](https://github.com/libass/libass)**: Portable subtitle renderer (ISC).
- **[libbluray](https://www.videolan.org/developers/libbluray.html)**: For Blu-ray disc support (LGPLv2.1).
- **[libdav1d](https://www.videolan.org/projects/dav1d.html)**: AV1 decoder (BSD 2-Clause).
- **[libdovi](https://github.com/quietvoid/dovi_tool)**: Dolby Vision metadata handling (MIT).
- **[MoltenVK](https://github.com/KhronosGroup/MoltenVK)**: Vulkan on Metal for macOS (Apache 2.0).

## macOS App Dependencies (Swift)
- **[Firebase iOS SDK](https://github.com/firebase/firebase-ios-sdk)**: Authentication and Realtime Database (Apache 2.0).
- **[Google Utilities & SDks](https://github.com/google)**: Supporting libraries for Firebase and metrics (Apache 2.0).
- **[Abseil](https://github.com/abseil/abseil-cpp)**: C++ library collection from Google (Apache 2.0).
- **[gRPC](https://grpc.io/)**: High-performance RPC framework (Apache 2.0).
- **[LevelDB](https://github.com/google/leveldb)**: Fast key-value storage (BSD-3-Clause).
- **[Swift Protobuf](https://github.com/apple/swift-protobuf)**: Apple's protocol buffer implementation (Apache 2.0).

- **[Nextra](https://nextra.site/)**: Next.js based site generator (MIT).
